Privacy Policy

This Privacy Policy applies to PuzzleDrop and other simple, non-commercial mini-games and hobby projects developed by us and hosted statically via GitHub Pages.

These projects are created solely for personal learning, portfolio presentation, and entertainment. They are entirely free to use, run on static environments, and do not carry advertisements, trackers, or commercial monetization.

🔒 Zero Data Collection Policy We do not collect, process, save, or share any personal details, metrics, or browsing histories. Because our games are built as static web apps, there is no database server logging or storing your data.

1. Third-Party Integrations

Certain dynamic modules within our games, like PuzzleDrop, let you search for or select images (via platforms like the Pinterest API) to construct personalized puzzles.

  • Local Processing: All connections to external image engines take place locally on your client browser.
  • No Retaining: We do not upload, register, or keep copies of your custom image URLs, search strings, or keys on any server infrastructure.
  • External Terms: Querying assets via third-party web protocols connects you with platforms operating under their own privacy standards. We advise reading their guidelines.

2. Tracking and Web Cookies

Our static web environments do not generate or track user cookies, analytical trackers (such as Google Analytics), or behavioral advertising identifiers.

3. Static Project Hosting

These portfolio projects run on GitHub Pages. To keep hosting channels reliable and secure, GitHub may process minor network transmission logs (like server request times and IP handshakes). This data is managed directly by GitHub under their global Privacy Statement.

4. Family & Children's Privacy

Because these applications require no registration and collect absolutely zero personal information, they are completely safe and clean for players of all ages, including children under 13.
